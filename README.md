# Power-Bi-Practice
📊 Project Title
Sales Performance Analysis: Stacked Bar Chart vs Stacked Column Chart Dashboard in Power BI

📖 Short Description / Purpose
This Power BI dashboard compares Stacked Bar Charts and Stacked Column Charts to analyze sales performance across multiple regions, categories, and sub-categories over the years 2023–2026. The dashboard helps users understand how different chart types can present the same data while providing valuable insights into yearly sales growth, regional performance, and category contributions.
The purpose of this project is to demonstrate effective data visualization techniques and support business decision-making through interactive reporting.

🎯 Objectives
Analyze yearly sales growth from 2023 to 2026. Compare sales performance across different regions. Evaluate category-wise and sub-category-wise sales contributions. Demonstrate the differences between Stacked Bar Charts and Stacked Column Charts. Create an interactive dashboard with dynamic filtering. Enable stakeholders to identify trends and growth opportunities.

📈 Key Insights
Regional Insights
West region consistently shows strong sales performance. East region demonstrates significant growth over time. Central region maintains stable sales.
South region contributes moderate sales levels.
Category Insights
Technology is one of the highest revenue-generating categories. Office Supplies maintain consistent performance. Furniture contributes steadily across all regions.
Growth Insights
Overall sales show positive growth from 2023 to 2026. Certain regions experience faster growth than others. Technology contributes significantly to yearly growth.
